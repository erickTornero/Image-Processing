## Requirements
* c++11
* Opencv 3.4
* ffmpeg

## Follow the instructions to execute Pattern Recognizion
### Install

Clone repository:

`git clone https://github.com/erickTornero/Image-Processing`

Open Right folder:

`cd Image-Processing/Pattern_Recognizion/`

Compile Project running the makefile:

`make`


### Execute the program:

`make exec`
